#### Setup

1. Clone the project

```bash
    git clone git@github.com:Emmanuel-Melon/kuwinda.git
```

2. Navigate to the project

```bash
    cd kuwinda
```

3. Choose Node.js version and install dependencies

```bash
    nvm use && yarn
```
